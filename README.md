# CeloHT

CeloHT is a Web3 public-interest platform built on three pillars:
- Education
- Agents (local economic actors)
- Reforestation

It leverages Celo cUSD and Valora for accessibility.

CeloHT is NOT a token, NOT a cryptocurrency, and conducts NO fundraising.

## Purpose
- Transparency
- Community audit
- Social impact

## Audit
All contracts are open-source and community-auditable.
